# Default structure for project
Default structure for the projects that I work:

###FOLDERS
``` 
├ name-project
    ├─ assets
        ├─ css
            -- app.css
            ├─ sass
                ├─ base
                ├─ FRAMEWORK
                ├─ generic
                ├─ objects
                ├─ partials                 
        ├─ fonts
            -- *.eot            
            -- *.svg            
            -- *.ttf            
            -- *.wof            
        ├─ js
            -- app.js            
            ├─ plugins
                -- *.js        
            ├─ project
                -- _*.js         
            ├─ requireds
                -- *.js
        ├─ images
            -- sprite.svg
            -- sprite.png
        ├─ svgs
            -- *.svg 
*.html
```
